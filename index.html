Data Analysis and Visualization

Analyzing data sets to uncover patterns, trends, and correlations using statistical methods and data visualization techniques. This involves using tools like Python, R, Tableau, Power BI, or Excel to create interactive dashboards and reports.

Web Design

Statistics

Designing and conducting hypothesis tests to assess the significance of relationships or differences in data. This includes tests such as t-tests, chi-square tests, ANOVA, and regression analysis.
